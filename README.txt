1) Install groovy (http://groovy.codehaus.org/Download)
2) run 'groovy -l <port> itsy-mvc.groovy"
3) point your browser to "http://localhost:<port>/"

controller.groovy - This is the Controller. You can use anything you like from the groovy language to collect data and return a map (the Model!) to the view.
view.html - A Groovy simple template (http://groovy.codehaus.org/Groovy+Templates) provides the View.
